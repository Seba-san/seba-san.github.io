campo.scripts.modularTree¶

La idea de este programa es que sirva para anotar las caracteristicas de cada opcion «importante» en modularTree addon. Funciona como una capa intermedia entre blender y el usuario final, algo asi como un traductor o un diccionario.

Un video donde se explica casi todo: Blender - Modular tree - tutorial en español
