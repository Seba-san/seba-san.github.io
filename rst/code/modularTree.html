No se que hace.

setAmount(amount=6)[fuente]¶

Cantidad de ramas, depende del espacio en el tronco principal, y de otras configs. No es exacto

setCanSpawnLeaf(val=True)[fuente]¶

Permite o no que estas ramas tengan hojas. (Puede pasar que el trongo haga un split, en ese lugar es raro ver ramas.) False or True

setFloorAvoidance(height=1)[fuente]¶

La rama mas baja, esta tan lejos del suelo como este parametro lo defina. Si por alguna otra config, las ramas estan todas arriba, este no le afecta. [0,+inf]

setGravityStrength(strength=1.1)[fuente]¶

Define que tanta fuerza hace la grabedad. El valor tipico es 0, pero puede ser -inf, +inf. Cuando es grande, simula que el arbol esta cargado, cuando es bajo hace que la mayoria de las puntas miren hacia arriba.

setLength(length=5.1)[fuente]¶

Longitud de cada rama (en promedio) expresado en metros?.

setRadius(radius=0.2)[fuente]¶

Determina el radio de las ramas como un porcentaje del radio del tronco principal [0,1] (va del 0% al 100%)

setSeed(val)[fuente]¶

Con los otros parametros iguales, genera cierta aleatoridad en la generacion del tronco

setSplitAngle(angle=1.0)[fuente]¶

Angulo con el que las ramas salen del tronco principal [0, 1.5]. 0 Es totalmente paralelas hacia arriba y 1.5 es perpendicular al tronco principal

setSplitProba(prob=1)[fuente]¶

Determina la probabilidad de que cada rama se vuelva a ramificar nuevamente (iterando sobre ella misma) [0,1]

setStart(start)[fuente]¶

Determina a partir de que altura salen las ramas. Esta expresado como porcentaje de la altura total del tronco principal [0,1] No chekeado
